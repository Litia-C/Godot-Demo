# 25-26-godot-game-Litia-C

DESCRIPCIÓN:

El juego consiste en meter conejos en una canasta, mientras más consigas meter, mayor será la puntuación. Juego hecho en Godot como prueba para trabajar con físicas y controles del ratón.

ESTRUCTURA:

Este proyecto se ha creado para que puede ejecutarse con Godot, así que en las carpetas he separado la música o efectos de sonido, lo assets visuales utilizados y el resto de archivos són el código del juego, las funciones de diferentes objetos.

DECISIÓN TÉCNICA:

He utilizado este lenguaje de programación porque ha tomado popularidad y estaba interesada en aprender a utilizar motores especializados en videojuegos, sobretodo ante la posiibilidad de migrar un proyecto a éste motor.

EJECUCIÓN:

Teniendo el motor de Godot, este proyecto importando todo el proyecto.
